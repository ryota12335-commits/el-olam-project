# El Olam Project

## Descripción
El Olam Project es un sistema web desarrollado para gestionar las operaciones de un grifo, permitiendo administrar ventas, inventario y reportes.

## Tecnologías utilizadas
- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## Instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/ryota12335-commits/el-olam-project.git
   ```

2. Instalar las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

3. Ejecutar la aplicación:
   ```bash
   python app.py
   ```
